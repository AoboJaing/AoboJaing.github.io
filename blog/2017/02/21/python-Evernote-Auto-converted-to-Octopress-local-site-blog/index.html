总结：

  1. \是一个特殊字符，它不能再字符串中正常的显示，如果必须显示，就这样写：\\
  2. DOS 命令里面指定文件路径时，只能使用\，不能使用/ 和 // ，使用这两个都是错的，都会导致 DOS找不到指定的文件路径
  3. 在DOS 命令里面，指定文件路径的 \ ，你写成 \\ 或者 \\\ 或者 \\\\\\ … 对是可以正常执行的，不会出现错误。
