order by — 以某个字段排序

例子：

总结：

  • order by a, b --a和b都是升序
  • order by a, b desc --a升序 b降序
  • order by a desc, b --a降序 b升序
  • order by a desc, b desc --a和b都是降序

文字描述：

  • 如果不制定排序的标准，则默认是升序。升序用asc表示，默认可以不写
  • 为一个字段制定的排序标准并不会对另一个字段产生影响
  • 强烈建议为每一个字段都指定排序的标准
