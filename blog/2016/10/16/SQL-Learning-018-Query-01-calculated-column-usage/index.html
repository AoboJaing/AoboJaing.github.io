注意： ： 在Oracle软件中字段的别名不允许用单引号括起来。但是 SQL Server 软件却允许，因此为了兼容性考虑，最好字段的别名用""（双引号）括起来，不要使用''（单引号）。

上面这段命令 select ename, sal*12 as "年薪" from emp 就是计算列。解释：原本 emp 表里面没有 年薪 这个属性，我们使用 sal*12 as "年薪" 得到年薪的信息，这就是 计算列。
