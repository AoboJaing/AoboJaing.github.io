聚合函数的分类

  • max() 求最大值
  • min() 求最小值
  • avg() 平均值
  • count() 求个数
    • count(*) ：返回表中所有的记录的个数
    • count(字段名) : 返回字段值非空的记录的个数， 重复的记录也会被当做有效记录
    • count(distinct 字段名) ：返回字段不重复并且非空的记录的个数
